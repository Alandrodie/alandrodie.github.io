So first we will create A Moving Edge Map for our current frame , to do so we will need some set of edges. To detect the edges we will use the Canny edge detector , which is a method that uses a Gradiant operation on a Gaussian convoluted image to show the edges of the image.

We will use 3 different set of edges : En which is the set of the edges detected by Canny on our current Image. Den which is the set of the edges detected from using Canny on the difference image (Canny (PreviousImage-CurrentImage) And finally Men-1 which the set of the moving edges on the previous image

With thoses 3 sets of edges we will create 2 moving Edge sets :

And MEstill which is the set generated by choosing edges that are still moving from our last image Moving Edge Map
To create the final Moving edge map we simply combine both of the previously created Moving Edge Map

Then we will extract the VOPs, for this we will find horizontal and vertical candidates, to do so we must simply by row and by column take the first and the last edge , everything in between will be considered a candidate.

In the created method , we didn't suceed at doing the vertical candidates , and because of this we couldn't continue further , but i will explain fastly what was supossed to happen next :
We needed to Combine both candidates with an AND logical operator , and then extract the VOPs thanks to morphological operations. After this we simply do a connected component analysis on the extracted VOPs and then compute the smoothness of motion trajectory (smoothness of direction and speed) finally an algorithm will decide if the VOP is "interesting" or not so we can choose the most intereseting segmented object to get .
